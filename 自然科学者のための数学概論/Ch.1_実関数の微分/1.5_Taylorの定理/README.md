# 1.5 Taylorの定理 - 視覚化ツール

このプロジェクトは、微分積分学における重要な定理の一つである「テイラーの定理」を幾何学的・直感的に理解するための視覚化ツールです。

## プログラムの意図

テイラーの定理は、複雑な関数を多項式で近似する手法ですが、数式だけでは「なぜ次数を上げると近似精度が上がるのか」や「剰余項（誤差）が何を意味しているのか」を理解するのが困難です。

本ツールは以下の3点を視覚化することを目的としています：
1.  **近似のプロセス**: 次数 $n$ を増やすことで、多項式 $P_n(x)$ が基準点 $a$ から順に元の関数 $f(x)$ へ重なっていく様子。
2.  **誤差の定量化**: 特定の点 $x$ における $f(x)$ と $P_n(x)$ の差（剰余項 $R_n(x)$）を垂直な線で示す。
3.  **ラグランジュの剰余項**: 誤差を完全に説明できる「中間点 $c$」が $a$ と $x$ の間に必ず存在するという、定理の核心部分（存在証明）を数値探索によって可視化する。

## ファイル構成

プロジェクトは関心の分離（Separation of Concerns）に基づき、以下の5つのファイルで構成されています。

| ファイル名 | 役割 |
| :--- | :--- |
| `index.html` | アプリケーションの骨格。p5.js ライブラリと各スクリプトを読み込みます。 |
| `sketch.js` | メインロジック。スライダーやマウスによる操作（インタラクション）と、描画のループを管理します。 |
| `math-utils.js` | 数学エンジン。高階数値微分、テイラー多項式の生成、ラグランジュ中間点 $c$ の探索など、核心となる計算を行います。 |
| `drawing-utils.js` | 描画ライブラリ。座標平面、グラフ、誤差を示す点線、補助ラベルなど、p5.js を用いた図形描画に特化しています。 |
| `config.js` | 設定ファイル。キャンバスサイズ、色のパレット、グリッドのスケールなど、見た目に関するパラメータを一括管理します。 |

## 操作方法

*   **次数スライダー（右下）**: 近似の次数 $n$ を 0（定数）から 9（高次）まで変更できます。
*   **点 a（緑）のドラッグ**: テイラー展開の基準点（展開の中心）を動かせます。
*   **点 x（紫）のドラッグ**: どの地点での近似精度を調べるかを指定できます。
*   **点 c（ピンク）**: ラグランジュの剰余項を成立させる中間点です。定理の予言通り、常に $a$ と $x$ の間に位置します。
