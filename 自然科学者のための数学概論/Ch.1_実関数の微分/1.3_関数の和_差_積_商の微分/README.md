# 1.3 関数の和・差・積・商の微分 - 視覚化ツール

本プロジェクトは、『自然科学者のための数学概論』の第1.3節「関数の和・差・積・商の微分」における数学的概念を、p5.jsを用いて動的に視覚化するための学習ツールです。

## スクリプトの意図と役割

### `sketch.js`（メイン・コントロール）
このファイルは、本視覚化システムの**エントリーポイントであり、学習内容の設定を行う中心的な役割**を担います。

- **関数の定義**: 視覚化したい具体的な関数（$f(x)$, $g(x)$）および、それらを組み合わせた関数（$f+g$ など）をここで定義します。教材の内容に合わせて、表示する関数を即座に変更できます。
- **描画パイプラインの制御**: グリッド、座標軸、そして複数のグラフを適切な順序で描画するよう制御します。
- **インタラクションの管理**: マウス操作による $x$ 座標の選択を検知し、その点における接線や微分係数をリアルタイムに計算・表示する橋渡しを行います。
- **重ね合わせの視覚化**: 複数の関数を異なる色で重ねて描画することで、「個々の関数の挙動」と「それらを合成した関数の挙動」の関係性を直感的に理解できるように構成されています。

## その他の構成ファイル

- **`math-utils.js`**: 微分（数値微分）や座標変換など、純粋な数学的計算ロジックを提供します。
- **`drawing-utils.js`**: p5.js を用いたグラフや座標平面の具体的な描画処理を担当します。関数を引数として受け取る汎用的な設計になっています。
- **`config.js`**: キャンバスサイズ、グリッドのスケール、配色などのデザインシステムを一括管理します。

## 使い方
1. `index.html` をブラウザで開きます。
2. 画面上をマウスでクリックまたはドラッグすると、その位置における接線と微分係数が表示されます。
3. `sketch.js` 内の関数 `f`, `g`, `h` を書き換えることで、和以外の演算（差、積、商）の性質も同様に視覚化できます。
