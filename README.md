# Creative Learning in Math Science

数学および自然科学（物理学など）の概念を、プログラミング（主に p5.js など）を用いて視覚的・インタラクティブに学び、表現するためのプロジェクト・学習記録リポジトリです。
理論や数式を「動くシミュレーション」として実装することで、より直感的で創造的な理解を目指しています。

## ディレクトリ構成

本リポジトリは、分野やテーマごとに以下のディレクトリに整理されています。

- **`1_linear_algebra/`**
  線形代数に関する視覚化やシミュレーション
- **`2_calculus/`**
  微分・積分学の概念の視覚化
- **`3_statics/`**
  静力学（力のつり合いやモーメントなど）に関するシミュレーション
- **`4_differential_equation/`**
  微分方程式の数値解法や力学系挙動の視覚化
- **`5_games/`**
  数学や物理の概念を応用したインタラクティブなプログラム・ミニゲーム
- **`6_sports/`**
  スポーツにおける物理挙動（放物運動・反発など）のシミュレーション
- **`7_leaning_angel/`**
  （テーマに沿ったプロジェクト）
- **`practice_p5_dom/`**
  p5.js での DOM 操作や UI 実装の練習用ディレクトリ
- **`自然科学者のための数学概論/`**
  寺沢寛一著『自然科学者のための数学概論』の章立てに沿い、概念を p5.js などを活用してプログラムとして実装・視覚化したディレクトリ

## 主な使用技術

- HTML5 / CSS3 / Vanilla JavaScript
- **p5.js** （描画・アニメーション・シミュレーションのコアとして活用）

## 目的

1. 抽象的な数式や定理をコードによって「動く形」に翻訳し、直感的な理解を深めること。
2. クリエイティブ・コーディングの手法を用いて、数学や科学の美しさを表現すること。
3. 『自然科学者のための数学概論』などの成書の内容を、現代のWeb技術で再解釈し、インタラクティブな教材として構築すること。
