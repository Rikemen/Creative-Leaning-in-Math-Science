# formetter (Prettier) の設定を1.5に追加した

## コードの整形 (Prettier)

本プロジェクトでは、コードの品質と一貫性を保つために [Prettier](https://prettier.io/) を導入しています。設定ファイル `.prettierrc` で定義されている主なルールは以下の通りです。

| オプション      | 設定値   | 意味                                               |
| :-------------- | :------- | :------------------------------------------------- |
| `printWidth`    | `100`    | 1行の最大文字数を100文字に制限（読みやすさの向上） |
| `singleQuote`   | `true`   | 文字列にシングルクォート (`'`) を優先的に使用      |
| `trailingComma` | `"none"` | 末尾のカンマを付与しない                           |
| `semi`          | `true`   | 文末に必ずセミコロン (`;`) を付与                  |

### 使用方法

Prettier 拡張機能(Prettier - Code formatter)をインストールし、保存時に自動整形されるように設定する。また.prettierrcで設定したいルールを書く。`.prettierignore` によって不要なファイルが整形対象から除外する。
