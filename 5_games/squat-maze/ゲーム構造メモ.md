public/js/
├── game.js ← エントリポイント（p5インスタンス生成のみ）
└── game/
├── constants.js ← 定数（色、サイズ、速度等）
├── player.js ← プレイヤーの描画・位置管理
├── walls.js ← 壁の生成・スクロール・描画
├── obstacles.js ← 障害物の生成・管理
├── collision.js ← 当たり判定ロジック
├── hud.js ← ライフ・スコア等のUI描画
├── effects.js ← CRTエフェクト・パーティクル等の演出
├── pose.js ← ml5.js 姿勢推定（カメラ取得・座標マッピング）
└── screens.js ← Ready / GameOver 画面の描画
