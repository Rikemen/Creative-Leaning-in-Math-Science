About

このページはテンプレートのサンプルページです。 新しいページを追加する際の参考として使ってください。

ページの追加方法

  1. このファイル（about.html）をコピーして新しいHTMLを作る
  2. app-header.js の navLinks 配列にエントリを追加する
  3. ページ固有のスクリプトがあれば public/js/ に配置する

Firestoreとの連携例

Firestoreを使う場合は、ページのスクリプトで firebase-config.js から db をインポートします。 

import { db } from './js/firebase-config.js';
import { collection, getDocs } from
  'https://www.gstatic.com/firebasejs/12.9.0/firebase-firestore.js';

const snapshot = await getDocs(collection(db, 'items'));
snapshot.forEach(doc => console.log(doc.data()));
