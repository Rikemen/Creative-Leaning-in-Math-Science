# ⚾ MVP定義: Baseball Form Analyzer

## コンセプト

> **「自分の投球動画をブラウザに読み込み、骨格を表示する」** — これだけをまず動かす。

Firebase認証・ストレージ・比較機能はすべて後回し。ローカルファイルだけで完結する最小構成。

---

## スコープ: やること / やらないこと

| ✅ やること (MVP)                          | ❌ やらないこと (Post-MVP)                    |
| ------------------------------------------ | --------------------------------------------- |
| ローカル動画ファイルの読み込み・再生       | Firebase認証・ユーザー管理                    |
| PoseNetによる骨格のリアルタイム描画        | 動画のアップロード・クラウド保存              |
| 再生/一時停止/フレーム送りの基本操作       | プロ選手との比較（オーバーレイ/サイドバイサイド） |
| 主要関節の角度表示（肩-肘-手首 etc.）     | 過去フォームとの成長記録                      |
| 単一HTMLファイルで動作                     | AIによるフォーム改善アドバイス                |

---

## 技術スタック (MVP)

| カテゴリ      | 採用技術         | 選定理由                               |
| ------------- | ---------------- | -------------------------------------- |
| **Frontend**  | p5.js (CDN)      | Canvas描画＋動画制御が1ライブラリで完結 |
| **ML Engine** | ml5.js (CDN)     | PoseNetのラッパー、セットアップ不要    |
| **構成**      | 単一HTML + JS    | ビルド不要、ブラウザで即実行           |

> **Firebaseは使わない。** サーバー構築もビルドツールも不要。

## 実装ステップ

### Step 1: 動画の読み込みと再生 🎬

- `index.html` に p5.js / ml5.js をCDNで読み込み
- ファイル入力(`<input type="file">`)で動画を選択
- `createVideo()` で動画をキャンバス上に描画
- 再生/停止ボタンの実装

### Step 2: PoseNetによる骨格検出 🦴

- `ml5.poseNet()` で動画を解析対象に設定
- 検出されたキーポイント（17点）をキャンバス上に描画
- 骨格線（肩→肘→手首 etc.）の描画

### Step 3: 関節角度の計算・表示 📐

- 3点間のベクトルから角度を算出するユーティリティ関数
- 投球に重要な角度（肩-肘、肘-手首、腰-膝）をリアルタイム表示
- フレーム送り機能で特定のポジションを確認可能に

## 完了条件 (Definition of Done)

- [ ] ブラウザ上で投球動画を読み込み、再生できる
- [ ] 動画上に骨格（関節点＋線）がリアルタイムで描画される
- [ ] 主要な関節角度が数値として画面に表示される
- [ ] フレーム単位でのコマ送り操作ができる

> **上記4つが動けば MVP 完了。** ここからFirebase統合・比較機能へ進む。
