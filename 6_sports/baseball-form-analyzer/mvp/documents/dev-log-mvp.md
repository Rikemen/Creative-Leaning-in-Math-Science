## Step 1: HTMLの骨格を作る（10分）

**ゴール:** ブラウザで空のページが表示される

- [x] `index.html` を作成
- [x] p5.js / ml5.js のCDNリンクを `<script>` で追加
- [x] `style.css` を作成し、リンク
- [x] `sketch.js` を空の `setup()` / `draw()` で作成
- [x] ブラウザで開いて、真っ白なキャンバスが表示されることを確認

```
作るファイル: index.html, style.css, sketch.js
確認方法: ブラウザで index.html を直接開く
```

**✅ 完了条件:** コンソールエラーなしでキャンバスが表示される

## Step 2: 動画ファイルの読み込み（10分）

**ゴール:** ファイル選択から動画を読み込み、キャンバスに表示される

- [x] 動画を読み込み、Canvas内にイメージとして表示する
- [x] 動画を非表示にし（`video.hide()`）、`image(video, ...)` でキャンバスに描画する
- [x] 動画を再生する
- [x] 動画のアスペクト比を保つようにする

**✅ 完了条件:** ローカルの動画ファイルがキャンバス内に表示される

## Step 3: 再生/停止ボタン（10分）

**ゴール:** 動画の再生と停止をボタンで制御できる

- [x] 再生/停止トグルボタンを `createButton()` で作成
- [x] `video.play()` / `video.pause()` をボタンに紐付け
- [x] ボタンのラベルを再生中は `⏸ 停止`、停止中は `▶ 再生` に切り替え

```
確認方法: ボタンを押して動画が再生/停止すること
```

**✅ 完了条件:** ボタンで動画の再生・停止が切り替わる

## Step 4: フレーム送り機能（10分）

**ゴール:** 1フレームずつコマ送り・コマ戻しができる

- [x] `◀ 前` / `次 ▶` ボタンを追加

- [x] ボタン押下でフレーム単位（約 1/30秒）の移動処理を実装
- [x] 動画が停止中のときだけコマ送りを有効にする

```
確認方法: 停止状態でコマ送りボタンを押し、1フレームずつ動くこと
```

**✅ 完了条件:** フレーム単位での前後移動ができる

## Step 8: 角度計算ユーティリティの実装（10分）

**ゴール:** 3つの関節点から角度を計算する関数が動作する

- [ ] `pose.js` に `calculateAngle(pointA, pointB, pointC)` を実装
- [ ] ベクトル演算（内積 / 外積）で2辺の間の角度をラジアン→度数に変換
- [ ] テスト用に固定座標値を入れて、期待する角度が返ることをコンソールで確認

```
確認方法: console.log で既知の角度（90°、180°等）が正しく算出される
```

**✅ 完了条件:** `calculateAngle()` が正しい角度を返す

## Step 9: 関節角度のリアルタイム表示（10分）

**ゴール:** 画面上に主要な関節角度が数値で表示される

- [x] 投球に重要な角度ペアを定義:

  - 腰-膝-足首（軸足の曲がり）
  - 肩-腰-膝（体の傾き）
- [x] 各フレームで `calculateAngle()` を呼び、結果をキャンバス下部に `text()` で表示
- [x] 角度の値を小数点1位に丸める

```
確認方法: 動画再生中に角度値がリアルタイムで変化すること
```

**✅ 完了条件:** 主要関節の角度がリアルタイムで数値表示される
